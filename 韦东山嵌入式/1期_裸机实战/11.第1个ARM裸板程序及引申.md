# 位操作
1. 移位  
左移：<<  
右移：>>
2. 取反  
~
3. 位与  
&  
4. 位或  
|  
***
# 置位操作
int a = 0x123;  
置位bit7 bit8  
int b = a | (1 << 7) | (1 << 8)
***
# 清位操作  
int a = 0x123;  
清位bit7 bit8  
int b = (a & ~(1 << 7)) & (~(1 << 8))
